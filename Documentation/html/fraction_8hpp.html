Detailed Description

Fraction Class file with definitions and declarations, (generally definitions and declarations should be separated in different files, here using templates to define/declare functions) Overloaded addition, subtraction,multiply,division,greater/lesser than,equality/non equality operators to perform appropriate actions with fractions.Also has display function and overloaded >> and << functions to show output and get input respectively. Also includes findRemainder and greatest Common divisor function.
